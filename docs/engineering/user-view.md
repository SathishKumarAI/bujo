# 👤 User view

What every view does, in plain language. (In-app: the top-bar `?` per view and
the ⓘ on each card say the short version; this is the full list.)

## The daily loop
1. **Today** — capture (bullet grammar `t/e/n · * · ^ · #tag`, or plain sentences
   via the smart bar), tick habits, rate mood/stress/sleep/energy, gratitude +
   memory. A **Coach** card shows "do this next" prompts from your data.
2. **Plan** — clear overdue tasks, define recurring routines, import `.ics`.
3. Review at night; carry forward what's unfinished.

## Views

| View | What you do there |
|---|---|
| **Today** | Daily log + capture, wellbeing sliders, coach, gratitude, memory |
| **Monthly** | Month calendar; events as dots, mood-tinted days |
| **Trackers** | Habit dot-grid; 3 lenses — **grid / activity / routine** (by time of day); per-habit notes |
| **Fitness** | Cardio + strength (tabs): logging, programs, nutrition, charts (deferred below logging) |
| **Pull-ups / Home Workout** | Dedicated bodyweight hubs with demos + programs |
| **Pickleball** | Sessions (rich logging), **leagues & tournaments**, **75-day 3.5→4.0 plan**, charts in the right rail |
| **Challenges** | 75 Hard / 90-day / custom disciplines |
| **Focus** | Deep-work / coding session tracker |
| **Reading** | Want / reading / finished shelves, page progress, ratings, "what I learned" log, **read-later links** |
| **Collections** | Future log, friends, birthdays, custom pages |
| **Goals** | Cross-view rollup of every active target |
| **Insights** | Streaks, correlations, search, weekly review, personal records |
| **Stats** | Charts wall; mood calendar + year-in-pixels (side-by-side or stacked), all ⛶-enlargeable |
| **Cycle / Streak** | Opt-in gendered wellbeing tools |
| **Account** | Real login / signup / guest / sign out (gated full-screen when signed out) |
| **Settings / Help** | Theme, units, data & cloud, passcode; in-app guide |

## Cross-cutting affordances
- **⌘K / Ctrl+K** — command palette (jump anywhere / run actions).
- **⛶** on chart cards — enlarge to a big modal.
- **ⓘ** on any card — what is this?
- **Onboarding tour** — one-time first-run guide.
- **Undo/redo** — ⌘Z / ⌘⇧Z, app-wide.

## Privacy promise
