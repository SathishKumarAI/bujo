## Cloud storage (already shipped)

Three local-first options in **Settings → Data & Cloud** — no backend, no account:

- **Own folder** (`lib/fscloud.ts`, File System Access API) — pick a folder inside
  your Drive/Dropbox/OneDrive sync; bujo auto-saves `bujo.json` there.
- **Google Drive** (`lib/gdrive.ts`) — opt-in, your own OAuth client.
- **GitHub gist** (`lib/github.ts`) — private-gist backup.

Plus JSON/Markdown export-import. The only deferred piece is **accounts +
E2E-encrypted multi-device sync** (needs a thin backend — out of the local-first
default; tracked as BUJO-91).
